== You can choose between 4 effect ==
*   Custom Post Type – Add your blog posts with custom post type.
*   Posts – Display your posts with custom styles.
*   Paginations – Get Posts with custom pagination style.
*   Categories, Tags – You can create your own categories and tags.

== Installation ==
1. Upload `netroics-blog-posts-grid` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy and paste <strong>if(function_exists('netroicstp_custom_post')){netroicstp__custom_posts_loop();}</strong> in template code or  <strong>[NetroicsPosts]</strong> in the post/page" to display Netroics Blog Posts with grid.
